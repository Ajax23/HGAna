    number¶

    hgana.affinity.number(data_link, temp, volume, num_mol=1)¶

    This function calculates the binding affinity \(\Delta G_N\). This is done through a brute force summation of all instances the cyclodextrin is occupied \(N_b\) and how often it is not \(N_u\). These numbers are then used to calculate the binding free energy

    \[\Delta G_N=-RT\ln\frac{N_b}{N_u}-RT\ln\frac{V}{NV_0}\]

    with number of solute molecules \(N\), standard gas constant \(R\), temperature \(T\), simulation box volume \(V\) and standard state volume \(V_0=1.661 \text{nm}^3\).

    Parameters:
    data_link : string

    Sampled affinity object

    temp : float

    Simulated temperature in \(\text{K}\)

    volume : float

    Simulation box volume in \(\text{m}^3\)

    num_mol : integer, optional

    Number of molecules
    Returns:
    table : DataFrame

    Pandas DataFrame containing the binding affinity calculated for the total number of instances and for both orientations. Values are given in \(\frac{\text{kJ}}{\text{mol}}\) and \(\frac{\text{kcal}}{\text{mol}}\)
